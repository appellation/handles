  • send(data: string, options?: IResponseOptions, messageOptions: object): Promise<Message | Message[]>
    • Defined in structures/Response.ts:83

    Send a message using the Discord.js Message.send method. If a prior response has been sent, it will edit that unless the force parameter is set. Automatically attempts to fallback to DM responses. You can send responses without waiting for prior responses to succeed.
